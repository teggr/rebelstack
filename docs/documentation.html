rebelstack Documentation

Documentation

Initial setup and dependency installation for a Rebelstack Maven project.

Install dependencies with the Rebelstack initializer

Use the JBang script from this repository to scaffold a Maven project with the full Rebelstack stack.

jbang scripts/rebelstack-init.java my-app
cd my-app
mvn spring-boot:run

Generated dependencies

<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-web</artifactId>
</dependency>
<dependency>
  <groupId>com.j2html</groupId>

  <version>1.6.0</version>
</dependency>
<dependency>
  <groupId>io.github.teggr</groupId>

  <version>0.1.0-SNAPSHOT</version>
</dependency>
<dependency>
  <groupId>org.webjars.npm</groupId>

  <version>2.0.4</version>
</dependency>

Deploy4j Maven plugin

<plugin>
  <groupId>com.github.teggr.deploy4j</groupId>

Notes

  • The initializer configures the JitPack repository for j2css.
  • Use jbang-catalog.json to register the script as an app in your own JBang setup.
