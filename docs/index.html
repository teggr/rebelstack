rebelstack Documentation

Full Java end-to-end web development

Rooted in web primitives, not abstractions.

Rebelstack combines Spring Boot, j2html, j2css, htmx and deploy4j so Java developers can build modern web applications with direct control over HTML, CSS and HTTP.

Get started View source

Why choose Rebelstack?

Server-rendered by default

Use Spring Boot + j2html for explicit HTML generation in Java. No heavy frontend framework required.

Typed styling in Java

Generate and compose CSS using j2css while still shipping standard browser-native stylesheets.

Progressive interactivity

Add dynamic behavior with htmx attributes and keep your app simple, fast, and HTML-first.

Built-in deployment path

Deploy consistently to self-hosted VMs with deploy4j and keep your release flow close to your app.

Stack components

  • Spring Boot — backend runtime and HTTP foundation
  • j2html — Java-first HTML rendering
  • j2css — typed Java CSS generation
  • htmx — modern browser interactions with minimal JS
  • deploy4j — deployment automation for Java web apps
